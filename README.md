# Weather API Application

## Overview

This is a Python-based Weather API Application that fetches real-time weather data using the Open-Meteo APIs.

The application allows users to enter a city name, retrieves its geographical coordinates using the Open-Meteo Geocoding API, and then fetches the current weather information using the Open-Meteo Weather API.

---

## Features

* Search weather by city name
* Real-time weather data
* Displays temperature
* Displays wind speed
* Displays day/night status
* Error handling for invalid city names
* Error handling for server and connection issues
